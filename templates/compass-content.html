Compass Core Framework
  • Design agnostic framework that provides a common code that would otherwise be duplicated across other frameworks and extensions
  • Variables (variables), Mixins (mixins), Functions (functions)
  • This file can be imported using - @import 'compass'
  • Basic Imports
    • CSS3
      • Cross browser mixins for css3 properties introduced in css3 like border-radius
      • @import 'compass/css3'
    • Typography
      • Basic mixins for common text styling patterns
      • @import 'compass/typography'
    • Utilities
      • Basic mixins for common styling patterns
      • @import 'compass/utilities'
  • Non-imported modules
      • Layout
        • Provides tools to help you with page layout
        • @import "compass/layout"
      • Reset
        • Adds a CSS Reset into your stylesheet
        • @import 'compass/typography'
